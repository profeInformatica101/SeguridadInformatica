TOR - The Onion Router

TOR es un software gratuito y de código abierto que permite una navegación anónima y segura por internet.

El nombre "The Onion Router" hace referencia a la estructura en capas que utiliza para ocultar la identidad del usuario.

Cómo funciona TOR

Cuando un usuario se conecta a TOR, su tráfico de internet se redirige a través de una red de servidores en todo el mundo, en lugar de seguir una ruta directa.

Estos servidores, llamados "nodos", encriptan el tráfico de datos varias veces y lo envían a través de diferentes nodos, hasta que finalmente llega al destino final. Esto hace que sea muy difícil para alguien interceptar el tráfico y rastrear su origen.


TOR Navegador normal
Privacidad Mayor privacidad ya que la conexión es enrutada a través de múltiples nodos, lo que dificulta la identificación del usuario. Menor privacidad ya que la conexión directa permite a los sitios web obtener la dirección IP y otros datos del usuario.
Velocidad Menor velocidad debido a que la conexión se enruta a través de múltiples nodos, lo que aumenta el tiempo de latencia. Mayor velocidad ya que la conexión es directa y no se enruta a través de nodos adicionales.
Accesibilidad Limitada ya que algunos sitios web pueden bloquear el acceso a los usuarios de TOR debido a su reputación de anonimato. Mayor accesibilidad ya que no hay restricciones de acceso a sitios web.
Seguridad Mayor seguridad ya que la conexión está encriptada y la dirección IP del usuario está oculta. Menor seguridad ya que la conexión no está encriptada y la dirección IP del usuario es visible.

Usos de TOR

TOR se utiliza a menudo para navegar por la web de forma anónima y acceder a sitios web bloqueados en ciertos países. También se utiliza para proteger la identidad de periodistas, activistas políticos y otras personas que necesitan comunicarse de forma segura y anónima.
